# Referências visuais de UI – ClicVend

Esta pasta deve conter as **imagens de referência** de UI para o projeto ClicVend. O documento principal de design system está em [../DESIGN-SYSTEM.md](../DESIGN-SYSTEM.md).

**Referência de interface:** Plataforma **digisac**. Copiar layout e componentes; aplicar logo ClicVend e paleta de cores do DESIGN-SYSTEM.md.

---

## Como usar

1. Coloque aqui as capturas de tela ou mockups que servem de modelo (telas da digisac ou equivalentes).
2. Use os nomes sugeridos abaixo para manter consistência com o DESIGN-SYSTEM.md.
