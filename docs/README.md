# Documentacao do Projeto

Este diretorio foi organizado por categoria para facilitar manutencao, busca e evolucao da documentacao.

## Estrutura

- `design/`: direcao visual, design system e referencias de UI.
- `features/`: documentacao funcional de modulos e fluxos do produto.
- `guides/`: guias operacionais (deploy, webhooks, ambiente, servidores).
- `legacy/`: documentos antigos/importantes para historico.
- `planning/`: planejamento, PRD, plano de implementacao, tarefas e criterios.
- `roadmap/`: roadmaps estrategicos de evolucao (campanhas, CRM comercial etc.).
- `scripts/`: espaco reservado para scripts/checklists auxiliares.
- `specs/`: especificacoes tecnicas e contratos (schema, OpenAPI etc.).
